# GHZ-State
This was my work in a practice lab for IBM's Qiskit Global Summer School 2025 for designing a GHZ state and running it on a quantum device. Credit for setting up this notebook belongs to IBM and this program.

What I learned:
How to set up a Greeberger-Home_Zeilinger (GHZ) state circuit and transpile it using a coupling map I designed based on the tutorial instructions. 

Installation & Usage:
To open this up, I used Ubuntu to create a Python environment. This was done by opening Ubuntu and placing the following commands:
1) python3 -m venv venv
2) source venv/bin/activate
3) jupyter lab --no-browser --ip=0.0.0.0 --port [insert a port number]

To run this, however, you may need to create a new runtime service, as this was designed to be for participants in the summer school program.
